Flat Button Icon Pack
This is a mobile-friendly flat buttons icons assets.

It include more than 8000 buttons and icons.

To reduce times to import to Unity, we zip some folders. 
You need unzip these folders before using them.


Features:

- PNG files and layered PSD files, 128x128 resolution; 

- 150 base icons in 14 color resulting in 2100 distinct icons;

- 4288 square buttons in 28 theme, 151 buttons for each theme;

- 4288 round buttons in 28 theme, 151 buttons for each theme;

- 28 rectangle buttons in 28 theme, 1 button for each theme;

- 28 oval buttons in 28 theme, 1 button for each theme;

- Both white and black icon sets to fit your application's needs.


- Theme0 for white buttons or icons, theme1 to theme13 for colorful buttons or icons;
- Color:red,orange,yellow,aqua,green,green,blue,cyan,midnightblue,blueviolet,purple,pink,grey,black.
